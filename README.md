# The Mind
# El juego de cartas The Mind es un juego en equipo donde todos tienen que colaborar para ordenar cartas de menor a mayor, pero sin poder hablar entre ellos. Cada jugador tiene algunas cartas y debe intentar adivinar el momento correcto para ponerlas, solo confiando en su intuición y en cómo siente el ritmo del grupo. A medida que avanzan, se vuelve más difícil, porque necesitan estar más conectados y entenderse sin decir nada.
